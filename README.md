# CreditWise Loan Approval System

## 📌 Overview
CreditWise is a supervised machine learning project designed to predict loan approval outcomes.  
It leverages multiple classification algorithms (KNN, Logistic Regression, Naive Bayes) and evaluates performance using precision, recall, and F1-score.

---

## ⚙️ Features
- End-to-end ML pipeline (data preprocessing → training → evaluation)
- Exploratory Data Analysis (EDA) with visualizations
- Feature engineering for improved model accuracy
- Binary classification for loan approval prediction
- Model comparison with key metrics (Precision, Recall, F1)

---

## 🛠️ Tech Stack
- **Languages:** Python  
- **Libraries:** scikit-learn, pandas, numpy, matplotlib, seaborn  
- **Environment:** Jupyter Notebook / Google Colab
